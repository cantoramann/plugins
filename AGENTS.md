# AGENTS.md

This file applies to all AI agents working in this repository. See `CLAUDE.md` for full project instructions.

## Rules

- **Commit ownership**: Never add Co-Authored-By, Signed-off-by, or any author/co-author trailer to commits. All commits belong to the user. Do not attribute work to an AI agent in commit metadata.
- **Secrets**: Never commit `.env` or hardcode credentials in tracked files.
- **Commit style**: Imperative mood, concise messages. No force pushes to main.
